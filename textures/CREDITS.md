# Texture Credits

All material textures in this folder are from **[ambientCG](https://ambientcg.com/)**,
released under **CC0 1.0 Universal (Public Domain)** — free for any use,
including commercial, with no attribution required.

Obtained as web-ready JPEGs via the MIT-licensed
[open3dFloorplan](https://github.com/theLodgeBots/open3dFloorplan) project.

| File | ambientCG source | Use |
|---|---|---|
| wood.jpg | WoodSiding009 | Wood furniture finish |
| walnut.jpg | WoodFloor043 | Dark wood |
| oak.jpg | WoodFloor040 | Light wood |
| marble.jpg | Tiles099 (porcelain) | Counters / stone look |
| tile.jpg | Tiles074 | Tile surfaces |
| concrete.jpg | Concrete048 | Concrete finish |
| brick.jpg | Bricks097 | Brick accent |
| stone.jpg | Rock051 | Stone accent |

These are applied as tiling color maps on a selected object in Design Studio 3D
(the "Material" restyle option). To add more, drop a `.jpg` here and add an
entry to the `MATERIALS` list in `js/designstudio3d.js`.
