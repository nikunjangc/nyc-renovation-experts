# 3D Model Credits

All furniture and fixture models in this folder are from the **Kenney Furniture
Kit**, released under **CC0 1.0 Universal (Public Domain)** — free for any use,
including commercial, with **no attribution required**. We credit the author
voluntarily because the work deserves it.

- **Author:** Kenney (kenney.nl)
- **Pack:** Furniture Kit — https://kenney.nl/assets/furniture-kit
- **License:** CC0 1.0 — https://creativecommons.org/publicdomain/zero/1.0/
- **Format:** glTF Binary (`.glb`), low-poly, web-optimized (4–56 KB each)

The optimized `.glb` files were obtained via the MIT-licensed
[open3dFloorplan](https://github.com/theLodgeBots/open3dFloorplan) project,
which redistributes the Kenney Furniture Kit in GLB form.

## Adding more models
1. Download CC0 models (e.g. [Poly Pizza](https://poly.pizza),
   [Quaternius](https://quaternius.com), more [Kenney](https://kenney.nl/assets)
   kits) as `.glb` and drop them in this folder.
2. Add an entry to `../data/catalog.json` with a `"glb": "models/yourfile.glb"`
   field (and a `"shape"` fallback + feet `w/d/h`).
3. Record the source + license here. Prefer CC0; if a model is CC-BY, it must
   be credited above.
